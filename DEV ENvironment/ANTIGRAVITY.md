# 소개
구글이 **제미나이 3 프로(Gemini 3 Pro)** 를 중심으로 구축한 **차세대 AI 에이전트 기반 통합 개발 환경(IDE)** 입니다. 
비주얼 스튜디오 코드(VS Code)의 포크(Fork)로 구축되었으며, 개발자가 복잡한 코딩 작업을 자율 AI 에이전트에게 위임할 수 있도록 설계되었습니다. 

현재 공개 미리보기 형태로 제공되어 무료로 사용이 가능합니다.

# 특징
• **모델 선택형 구조:** Gemini 3 Pro뿐만 아니라 앤트로픽의 클로드(Claude) 4.5, 일부 GPT 계열 모델 등 다양한 최신 모델을 선택하여 사용할 수 있습니다.
• **에이전트 중심 개발:** 에이전트가 편집기, 터미널, 브라우저를 동시에 제어하여 코드 작성, 패키지 설치, 브라우저 테스트까지 한 번에 수행합니다.
• **병렬 작업 및 아티팩트 생성:** 여러 에이전트가 병렬로 작업할 수 있으며, 사용자의 신뢰를 위해 작업 목록, 구현 계획, 스크린샷 등의 검증 가능한 '아티팩트'를 생성합니다.
• **멀티 플랫폼 지원:** Windows, macOS, Linux 환경에서 모두 사용 가능합니다


# 사용법 및 워크플로우
1. **설치 및 초기 설정:** 공식 사이트에서 설치 파일을 내려받아 설치한 후, 기존 VS Code나 커서(Cursor)의 설정을 그대로 가져올 수 있습니다.

2. **개발 모드 선택:** 사용자의 필요에 따라 에이전트가 주도하는 모드(**Agent-driven**), 에이전트의 보조를 받는 모드(**Agent-assisted**), 또는 코드 리뷰 모드(**Review-driven**) 중 하나를 선택합니다.

3. **프롬프트 입력:** "더 깔끔하고 모던하게 리디자인해달라"와 같은 간단한 명령어를 입력하면 에이전트가 전체 구현 플랜을 세웁니다.

4. **실행 및 승인:** 에이전트가 생성한 플랜이 마음에 들면 "Proceed" 버튼을 눌러 작업을 진행하며, 코드 변경 시마다 우측 하단의 "Accept" 버튼을 눌러 검토 후 승인합니다.

5. **브라우저 연동 테스트:** 에이전트가 브라우저 확장 프로그램을 통해 자동으로 스크롤, 클릭 등을 수행하며 디자인과 반응형 동작을 실시간으로 테스트합니다.

# 모드별 차이점
**1. 에이전트 주도 개발 (Agent-driven development)**

• **특징:** 에이전트에게 **작업의 거의 전권을 맡기는 모드**입니다.

• **작동 방식:** 에이전트가 편집기, 터미널, 브라우저를 동시에 제어하며 코드 작성부터 패키지 설치, 브라우저 테스트까지 자율적으로 수행합니다. 사용자는 에이전트가 세운 계획을 승인(Proceed)하고 결과물을 검토(Accept)하는 역할에 집중합니다.

**2. 에이전트 보조 개발 (Agent-assisted development)**

• **특징:** **사용자가 주도적으로 코딩**을 진행하되, AI 에이전트가 **보조적인 도움**을 주는 모드입니다.

• **작동 방식:** 사용자의 코딩 흐름을 방해하지 않으면서 필요한 순간에 코드 추천이나 부분적인 구현을 도와줍니다.

**3. 리뷰 중심 개발 (Review-driven)**

• **특징:** 사용자가 작성한 **코드의 품질을 검증**하는 데 특화된 모드입니다.

• **작동 방식:** 사용자가 짠 코드를 에이전트가 분석하여 버그를 찾거나, 리팩토링 제안 및 코드 리뷰를 수행합니다.

**4. 작업 계획 모드 (Planning mode)**

• **특징:** 실제 코드를 작성하기 전, **심층적인 분석과 로드맵 수립**을 우선시하는 모드입니다.

• **작동 방식:** 주로 **Gemini 3 Pro High 모델과 조합**하여 사용하며, 에이전트가 브라우징을 통해 전체 페이지 구조를 파악한 뒤 구체적인 구현 단계를 포함한 '아티팩트(작업 목록, 계획 등)'를 생성합니다. 소위 '바이브 코딩'처럼 추상적인 의도를 구체적인 실행 계획으로 바꿀 때 유용합니다.

**5. 빠른 모드 (Fast mode)**

• **특징:** 복잡한 계획 단계보다는 **빠른 작업 처리와 피드백**에 최적화된 워크플로를 제공합니다.

**6. 커스텀 (Custom)**

• **특징:** 위의 여러 모드를 사용자의 필요에 따라 **조합하여 세밀하게 커스터마이징**할 수 있는 옵션입니다.

이 외에도 안티그래비티는 **관리자 보기(Admin view)**를 통해 여러 에이전트가 병렬로 작업하는 과정을 한눈에 모니터링하고 오케스트레이션할 수 있는 기능을 제공하여 개발 효율성을 극대화합니다.

# 실전 바이브코딩 예시
제미나이 3 프로를 활용한 바이브 코딩의 대표적인 실전 예시는 다음과 같습니다.

• **한 줄 프롬프트를 이용한 리디자인:** 기존의 강의 커뮤니티 랜딩 페이지를 **"더 깔끔하고 모던하게" 리디자인해달라**는 간단한 명령어만 입력합니다.

• **자율적인 페이지 분석 및 계획 수립:** 에이전트는 별도의 디자인 설명이나 구조 설명 없이도 **통합 브라우저 기능을 통해 전체 페이지를 스스로 이해**하고, HTML, CSS, JavaScript 파일 생성 및 구체적인 구현 단계를 포함한 플랜을 자동으로 도출합니다.

• **한국어 구현 플랜 확인:** 에이전트가 먼저 영어로 작성한 리디자인 플랜을 요청에 따라 **한국어로 즉시 변환**하여 사용자에게 제시하며, 사용자는 "Proceed" 버튼 하나로 전체 작업을 승인할 수 있습니다.

• **실시간 코드 생성 및 자동 테스트:** 에이전트가 코드를 작성함과 동시에 **브라우저를 자동으로 조작(스크롤, 클릭 등)**하여 레이아웃과 반응형 동작을 직접 테스트하고 검증합니다.

• **사용자 검토 및 승인:** 코드 변경 시마다 화면 우측 하단에 뜨는 **'Accept' 버튼**을 통해 사용자가 실시간으로 변화를 확인하고 승인하는 워크플로가 진행됩니다.

• **완성도 높은 결과물 도출:** 그라데이션 배경과 정리된 타이포그래피가 적용된 모던한 스타일은 물론, **숫자·지표 데이터, 로드맵 섹션, FAQ 섹션** 등을 추가로 작성하여 웹페이지의 콘텐츠 밀도를 높여줍니다.

이러한 방식은 코딩이나 디자인 지식이 부족한 비개발자도 **말로만 요구사항을 설명하고 결과물을 보며 수정을 반복**하는 것만으로 완성도 높은 웹페이지를 뽑아낼 수 있게 해줍니다. 바이브 코딩처럼 깊은 플래닝이 필요한 경우에는 **Gemini 3 Pro High 모델과 Planning 모드**를 조합하여 사용하는 것이 권장됩니다
