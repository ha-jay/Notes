# 에디터 실행법
터미널에 `nano`와 파일명을 입력하면 됩니다.

- **새 파일 만들기 또는 기존 파일 열기:** `nano filename.txt`
    
- **특정 경로의 파일 열기:** `nano /path/to/directory/filename.txt`

- 줄 번호 표시하기 : `nano -c filename.txt`
	- 에디터 내부에서 줄번호 표시 : `Alt + Shift + 3` (# 표시)
# 필수 단축키
나노에서 `^` 기호는 **Ctrl** 키를 의미합니다. (예: `^O`는 `Ctrl + O`)

|**기능**|**단축키**|**설명**|
|---|---|---|
|**저장하기**|`Ctrl + O`|'Write Out'. 파일 이름을 확인하고 `Enter`를 누르면 저장됩니다.|
|**종료하기**|`Ctrl + X`|에디터를 나갑니다. 수정사항이 있다면 저장 여부를 묻습니다.|
|**잘라내기**|`Ctrl + K`|현재 커서가 있는 한 줄을 통째로 삭제(잘라내기)합니다.|
|**붙여넣기**|`Ctrl + U`|잘라낸 내용을 현재 위치에 붙여넣습니다.|
|**검색하기**|`Ctrl + W`|'Where Is'. 찾고 싶은 단어를 입력합니다.|
|**도움말**|`Ctrl + G`|전체 단축키 목록을 보여줍니다.|
