 # AI를 학습파트너로 활용하는 방법
 AI를 활용하여 공부를 하고싶은데, AI를 활용해서 복붙만 하다보면, 내가 뭘하고있는건지, 마치 <mark style="background:#d3f8b6">자동사냥을 켜놓고 보고있는듯한 착각</mark>이 든다. 

단순히 코드를 그냥 받아쓰면 실력이 늘지 않는다. 
AI는 분명히 좋다 하지만 어떻게 해야할까? 
<mark style="background:#d3f8b6">학습의 주도권을 내가 쥐어야 한다.</mark> 
학습의 주도권을 내가 쥐고, 좀더 원활한 이해를 위해서 AI를 보조수단으로 사용해야한다. 


 ## 학습의 주도권을 쥐고 공부하려면 어떻게 해야하는가?
 > 도구 : 알아야 할것들에 대한 것을 파악한다. (책, 유투브, 커리큘럼)
 > 학습 : 교재, 유투브, AI활용
 > 모르는것 : AI활용 
 
 <mark style="background:#d3f8b6">AI TOOLS : 대화형 AI - cluad, gemini, chatgpt</mark>

### 1. 교재나 유투브 활용하여 학습
먼저 교재나 유투브를 활용하여 학습을 진행한다. 

### 2. 학습 중간에 AI에게 모르는것을 바로바로 질문한다. 
```
좋은 질문 예시:

"JavaScript에서 비동기가 뭔지 모르겠어. 
 초등학생한테 설명하듯이 쉽게 알려줘"

"이해했어. 그러면 실제로 어디에 쓰여?"

"Promise랑 async/await 차이를 코드 예시로 보여줘"

"이 코드에서 왜 await을 써야 해? 안 쓰면 어떻게 돼?"
```

**포인트:** 책은 질문을 못 받지만 AI는 받습니다. 이해될 때까지 계속 물어보세요. 부끄러울 필요 없어요.

### 직접 만들며 ai에게 도움받기

튜토리얼만 클론코딩만 하면 금방 잊어버립니다. 
<mark style="background:#d3f8b6">**직접 뭔가를 만들면서 배우는 게** 훨씬 효과적</mark>이에요.

**추천 방식 — "만들고 싶은 것"을 먼저 정하기:**

```
웹 프론트엔드  →  나만의 포트폴리오 페이지, 날씨 앱
백엔드         →  간단한 메모 저장 API
앱             →  할일 목록 앱
AI/데이터      →  CSV 파일 분석 스크립트
```

그리고 이렇게 접근합니다:

```
1. 일단 혼자 해보기  →  막힘
2. AI한테 힌트만 요청  →  "어떤 방향으로 접근하면 될까?"
3. 그래도 모르면  →  "이 부분 코드로 보여줘, 근데 왜 이렇게 하는지 설명도 해줘"
4. AI 코드를 받으면  →  한 줄씩 이해하고 직접 다시 쳐보기
```

**절대 하면 안 되는 것:** AI 코드를 그냥 복붙하고 넘어가기

### 에러를 AI로 디버깅하기

<mark style="background:#d3f8b6">**도구: Claude**</mark>

에러가 나면 그냥 AI한테 던지지 말고, 이렇게 활용하세요:

```
나쁜 방법:
"에러 났어" + 코드 붙여넣기 → AI가 고쳐줌 → 이해 못함

좋은 방법:
1. 에러 메시지를 직접 읽어보기 (30초만이라도)
2. 뭘 모르는지 파악하기
3. "이 에러 메시지가 무슨 뜻이야?" 먼저 물어보기
4. 왜 이런 에러가 났는지 이해한 후 해결
```

에러를 직접 이해하는 게 가장 빠르게 실력 느는 순간입니다.

###  코드 리뷰 받기

**도구: Claude**

뭔가 만들고 나면 AI한테 피드백을 받으세요.

```
"이 코드 내가 짰는데, 더 나은 방법이 있을까?
 초보자 관점에서 리뷰해줘"

"이 코드 읽기 좋게 개선하면 어떻게 돼?"

"실무에서는 이런 상황에서 보통 어떻게 해?"
```

혼자 공부해도 코드 리뷰를 받을 수 있다는 게 AI 시대의 가장 큰 장점입니다.

## 분야별 시작점 추천

튜토리얼 몇 개 해봤다면, 이렇게 순서를 잡는 게 좋습니다.

**웹 프론트엔드** → HTML/CSS 복습 후 JavaScript → React 순서로, 각 단계마다 작은 페이지 하나씩 만들기

**백엔드** → Node.js + Express로 간단한 API 만들기부터. "TODO 앱 API" 하나 만들어보면 핵심 개념이 다 들어있어요

**앱 개발** → 웹을 어느 정도 알면 React Native가 진입장벽이 낮습니다

**AI/데이터** → Python 기초 → pandas로 데이터 다루기 순서. 수학 몰라도 일단 시작할 수 있어요
