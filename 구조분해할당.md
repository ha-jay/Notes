# 구조분해할당
객체나 배열의 값을 해체하여 개별 변수에 즉시 할당하는 ES6 문법입니다.

### 1. 객체 구조 분해 (Object Destructuring)

변수 이름과 객체의 키값을 매칭합니다.
- **기본:** `const { name, age } = user;`
- **이름 변경:** `const { name: userName } = user;` (name을 userName이라는 변수로 사용)
- **중첩 객체:** `const { settings: { theme, lang } } = user;`
    

### 2. 배열 구조 분해 (Array Destructuring)

배열의 인덱스 순서에 따라 값을 할당합니다.

- **기본:** `const [a, b] = [10, 20];`
    
- **건너뛰기:** `const [first, , third] = [1, 2, 3];` (두 번째 요소 무시)
    

### 3. 나머지 매개변수 (`...rest`)

- 특정 요소를 제외한 나머지를 다시 배열로 묶을 때 사용합니다.
    
- **주의:** 반드시 마지막 위치에만 올 수 있습니다. `[...rest, last]` (X)
    
- `const [a, b, ...others] = [1, 2, 3, 4, 5];` → `others`는 `[3, 4, 5]`

### 4. 함수 매개변수 활용

함수로 전달되는 객체나 배열을 인자 단계에서 바로 분해하여 가독성을 높입니다.

```js
// 기존 방식
function printUser(user) { console.log(user.name); }

// 구조 분해 할당 방식
const printUser = ({ name, age }) => {
  console.log(`${name}은 ${age}세입니다.`);
};
```

### 💡 팁 - 변수없을경우 대비 기본값 설정

구조 분해 할당을 사용하면 변수가 존재하지 않을 경우를 대비해 **기본값**을 설정할 수도 있습니다.

- `const { name = 'Anonymous' } = user;` (user에 name이 없을 때 'Anonymous' 사용)
